Service questions

Questions about working together

Which AI automation service should I start with?

Most businesses start with the $497 AI Starter Assessment. It identifies your highest-value automation opportunities and includes one quick-win automation, so you see results before committing to a larger build. From there you can move to a Done-For-You Deployment or a Monthly Partner plan.

What is included in the Done-For-You Deployment?

It includes end-to-end automation of one core workflow, AI agents connected to your real tools, full testing, documentation, and a team handoff, plus 30 days of post-launch support. Pricing is $2,500–$4,500 one-time, based on the number of steps and integrations.

How long does an AI automation project take?

A Starter Assessment is usually delivered within one week. A Done-For-You Deployment typically takes one to three weeks depending on complexity. Monthly Partner clients receive new automations on a rolling basis each month.

Can you work with the tools I already use?

Yes. We build automation that connects to tools you already use — email, spreadsheets, CRMs, billing systems, and scheduling software — using platforms like Make.com and Cloudflare Workers. You rarely need to switch software.
